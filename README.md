# vue-touchable

> 仿IOS原生触摸反馈效果

## 安装

``` bash
npm install vue-touchable
```

## 使用

``` js
import { TouchableOpacity, TouchableHighlight } from 'vue-touchable'

<template>
  <touchable-opacity>
    <div>
      Click to toggle opacity
    </div>
  </touchable-opacity>
</template>
```

## dev

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev
```
